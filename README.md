# EnergyHome AI

EnergyHome AI vous permet d'optimiser la consommation 
électrique pour la conformité de votre foyer par rapport 
à la loi Climat avec l'intelligence artificielle.

## Technologies utilisées
- HTML5 / CSS3
- Media Queries
- Font Awesome
- Google Fonts

## Fonctionnalités
- Landing page responsive
- Header fixe
- Section "Comment ça marche"
- Footer professionnel
